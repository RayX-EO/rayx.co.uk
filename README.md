# RayX.co.uk (static site)

This site positions RayX as an **Earth Observation product suite**:

- **SIAC** — sensor-invariant surface reflectance (CEOS ARD-aligned expectations)
- **ARC** — agriculture foundation models (light → crop states)
- **Yield** — yield intelligence (estimation + early prediction)

## Deploy to GitHub Pages
1. Copy files into the repo root.
2. Commit + push to `main`.
3. GitHub → Settings → Pages → Deploy from branch → `main` / root.

## Customize
- Update the email in `contact.html`.
- Replace images under `assets/img/`.
