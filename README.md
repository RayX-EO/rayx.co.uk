# RAYX Website (GitHub Pages)

This is a simple static landing page for **RAYX**.

## Deploy on GitHub Pages
1. Push this repo to GitHub.
2. Go to **Settings → Pages**.
3. Set Source to **Deploy from a branch**, select `main` and `/ (root)`.

## Custom domain: rayx.co.uk
- In **Settings → Pages**, set the custom domain to `www.rayx.co.uk` (recommended).
- Add DNS records at your domain registrar:
  - `CNAME` for `www` → `<your-username>.github.io`
  - Optional: redirect root `rayx.co.uk` → `www.rayx.co.uk`
